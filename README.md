## How to use

<p>
Clone this repository to your system:
</p>

```
git clone https://github.com/seu-usuario/repository-name.git
```

<p>
Navigate to the repository directory:
</p>

```
cd repository name
```

<p>
Give the command to start the game menu:
</p>

```
python main.py
```
